# 🐍 Snake Game (Python + Pygame)

A classic Snake Game built using Python and Pygame with smooth controls and sound effects.

---

## 🎮 Features
- Smooth snake movement
- Score system
- Sound effects (eat & game over)
- Game over detection

---

## 🛠️ Tech Stack
- Python
- Pygame

---

## 🚀 How to Run
pip install pygame
python snake.py

---

## 📁 Project Structure
SnakeGame/
 ├── snake.py
 ├── sounds/
 │    ├── eat.wav
 │    └── gameover.wav
